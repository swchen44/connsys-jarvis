# Rubric: Skill Create Flow (FW-001)

## 評分標準 (1-10)

### 9-10 分：優秀
- SKILL.md 完整且格式正確
- 包含所有必要的 frontmatter 欄位（name, description, version, allowed-tools）
- 目錄結構符合 connsys-jarvis 慣例
- 附帶 README.md 說明
- 名稱使用 kebab-case

### 7-8 分：良好
- SKILL.md 存在且基本正確
- frontmatter 包含 name 和 description
- 目錄結構基本正確

### 5-6 分：及格
- SKILL.md 存在但格式有缺漏
- 缺少部分 frontmatter 欄位

### 3-4 分：不及格
- SKILL.md 存在但格式錯誤
- 重要欄位缺失

### 1-2 分：失敗
- 未能建立 SKILL.md
- 或建立了完全錯誤的結構
