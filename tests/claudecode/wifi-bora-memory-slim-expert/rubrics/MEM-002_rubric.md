# Rubric: Memory Slim Flow (MEM-002)

## 評分標準 (1-10)

### 9-10 分：優秀
- 完整的 ROM/RAM 分析報告
- 包含 section 分佈表格
- 列出 top symbols 及大小
- 提供具體可執行的優化建議
- 正確使用 memslim-flow skill
- 數字有單位且格式清楚

### 7-8 分：良好
- ROM/RAM 分析基本完整
- 有 section 或 symbol 資訊
- 有優化建議（即使不夠具體）
- 成功觸發 memslim-flow

### 5-6 分：及格
- 有 ROM 或 RAM 的部分分析
- 數字存在但不夠完整
- 建議較為籠統

### 3-4 分：不及格
- 只有片段資訊
- 未能成功執行分析命令
- 無優化建議

### 1-2 分：失敗
- 未能觸發 memslim-flow
- 或完全無法提供記憶體分析
