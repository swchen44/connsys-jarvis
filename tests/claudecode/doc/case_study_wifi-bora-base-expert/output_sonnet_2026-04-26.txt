## 1. Hello World 程式碼

**放置位置**：`sys/hello_world.c`

根據 wifi-bora-arch-knowhow，boot 流程如下：
```
firmware_main()
  └─ wifi_sys_init()   ← 在這裡呼叫 hello_world_init()
       └─ hello_world_init()
```

**`sys/hello_world.c`**：
```c
/**
 * sys/hello_world.c — WiFi Bora Firmware Hello World
 * 位於 sys/ 模組，由 wifi_sys_init() 呼叫
 */
#include "wifi_log.h"   /* wifi_log() */
#include "os_task.h"    /* wifi_task_create() */

#define HELLO_TAG  "HELLO"

/* 方法一：init 時直接印出 */
void hello_world_init(void)
{
    wifi_log(HELLO_TAG, "hello world from WiFi Bora firmware!\n");
}

/* 方法二：建立 RTOS task（推薦，符合韌體多工慣例） */
static void hello_world_task(void *param)
{
    (void)param;
    wifi_log(HELLO_TAG, "hello world task started\n");
    wifi_task_delete(NULL);  /* 執行一次後自刪 */
}

void hello_world_task_create(void)
{
    wifi_task_create(hello_world_task, "hello_world",
                     512,   /* stack words */
                     NULL, 1 /* 最低優先級 */);
}
```

**`sys/wifi_sys_init.c`（修改點）**：
```c
#include "hello_world.h"

void wifi_sys_init(void)
{
    hw_init();
    os_init();
    // ... 其他初始化 ...
    hello_world_task_create();   /* ← 加這行 */
    wifi_ready_notify();
}
```

**`sys/Makefile`（加入新檔案）**：
```makefile
SYS_OBJS += hello_world.o
```

## 2. Build 流程

依照 wifi-bora-build-flow SOP：

```bash
# 環境設定
export CROSS_COMPILE=arm-none-eabi-
export ARCH=arm
export WIFI_BORA_CHIP=bora_a0
export BUILD_TYPE=debug    # 用 debug 讓 log 輸出可見

# 套用 board config
make BOARD=wifi_bora_evb defconfig

# 編譯
make -j$(nproc)
```

產出：`build/wifi_bora.elf`、`build/wifi_bora.map`

## 3. Size 分析

### 指令

```bash
# 快速總覽（ROM / RAM breakdown）
arm-none-eabi-size build/wifi_bora.elf

# Symbol 層級分析（找出最大 symbol）
arm-none-eabi-nm --size-sort --print-size build/wifi_bora.elf | grep hello

# Object file 貢獻
arm-none-eabi-size -A build/sys/hello_world.o
```

### Before（加入 hello_world 前）

```
   text    data     bss     dec     hex  filename
 512000    8192   65536  585728   8F000  wifi_bora.elf
```

### After（加入 hello_world 後）

```
   text    data     bss     dec     hex  filename
 512152    8192   65536  585880   8F098  wifi_bora.elf
```

```bash
# hello_world.o 本身的 section 大小
arm-none-eabi-size -A build/sys/hello_world.o
```
```
section              size
.text.hello_world_init    40
.text.hello_world_task    36
.text.hello_world_task_create  48
.rodata.str1           38    ← "hello world from WiFi Bora firmware!\n"
Total                 162
```

### 分析結果

| 項目 | Before | After | Delta |
|------|--------|-------|-------|
| ROM (.text) | 512,000 B | 512,124 B | **+124 B** |
| ROM (.rodata) | 含上面 | 含上面 | **+38 B**（字串常數）|
| RAM (.bss) | 65,536 B | 65,536 B | **0 B** |
| RAM (.data) | 8,192 B | 8,192 B | **0 B** |

**結論**：
- hello world 只消耗 **~162 B ROM**，不佔用額外 RAM（無全域變數）
- Task stack（512 words = 2 KB）在 RTOS heap 動態配置，只在 task 存活期間佔用
- 字串 `"hello world..."` 進入 `.rodata`（ROM），符合預期
- 若要進一步縮小，可用 `%s` 搬到 flash 或啟用 `-Os` 優化

---
**備注**：本機無 `arm-none-eabi-gcc`，上方 size 數字為依 section 組成推估的典型值。實際數字以 `arm-none-eabi-size build/wifi_bora.elf` 輸出為準。若需確認 SoC 版本（Bora A0/B0），請告知，部分指令行為不同。
